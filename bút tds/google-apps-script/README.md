# Ket noi Google Sheet

1. Tao mot Google Sheet moi.
2. Vao Extensions > Apps Script.
3. Dat noi dung trong `Code.gs` vao file Apps Script.
4. Bam Deploy > New deployment.
5. Chon type la Web app.
6. Execute as: Me.
7. Who has access: Anyone.
8. Bam Deploy va lay Web app URL ket thuc bang `/exec`.
9. Gui URL do de cau hinh bien `GOOGLE_SHEETS_WEB_APP_URL` tren Cloudflare Pages.

Sau khi cau hinh xong, don hang se duoc ghi vao tab `DonHang`.
